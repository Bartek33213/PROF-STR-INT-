Profesjonalna, dynamiczna aplikacja webowa służąca do szacowania całkowitych kosztów sprowadzenia pojazdu ze Stanów Zjednoczonych do Europy. Projekt został stworzony w ramach zaliczenia z wykorzystaniem czystego HTML, CSS oraz JavaScript.

## 🚀 Główne funkcjonalności
* **Dynamiczne obliczenia:** Przeliczanie kosztów w czasie rzeczywistym bez przeładowywania strony (DOM manipulation).
* **Złożona logika matematyczna:** Uwzględnienie zmiennych takich jak: pojemność silnika (wpływ na akcyzę) oraz docelowy port odprawy celnej (różnice w stawkach VAT między Polską a Niemcami).
* **Responsywny interfejs (RWD):** Nowoczesny wygląd oparty na technologii Flexbox, dostosowujący się do urządzeń mobilnych.

## 🛠 Wykorzystane technologie
* **HTML5** - semantyczna struktura dokumentu.
* **CSS3** - stylowanie, Flexbox, zmienne CSS (`:root`).
* **JavaScript (Vanilla JS)** - logika obliczeniowa i interaktywność.
* **Git / GitHub** - system kontroli wersji.
